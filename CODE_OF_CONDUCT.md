# Code of Conduct

## Our Pledge
We want this project to be a welcoming, respectful, and constructive place for collaboration.

## Expected Behavior
- be respectful and professional
- assume good intent
- give actionable feedback
- focus criticism on ideas, code, and documentation, not people
- help keep the project useful for contributors with different levels of experience

## Unacceptable Behavior
- harassment or personal attacks
- discriminatory language
- bad-faith disruption
- doxxing or sharing private information without permission

## Enforcement
Project maintainers may remove, edit, or reject contributions and interactions that do not align with this code of conduct.
