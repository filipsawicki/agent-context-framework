## Short Description
File-based framework for bootstrapping, preserving, and handing off project context across AI agent sessions.

## About / Long Description
Agent Context Framework (ACF) helps teams keep continuity in AI-assisted software projects.

It combines:
- canonical Markdown context files in `context/*`
- a repeatable workflow for session startup and handoff
- MCP Memory as a supporting recall layer
- helper scripts for setup, verification, and MCP operations

Use it as a starter or shared project framework when you want agent sessions to stay aligned across resets, compaction, and handoffs.
